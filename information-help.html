Information about the website

This website provides information and resources to help users understand the features and functionalities available. The website was created so that you could properly and easily manage your daily tasks and activities.

Αυτός ο ιστότοπος παρέχει πληροφορίες και πόρους που βοηθούν τους χρήστες να κατανοήσουν τις διαθέσιμες δυνατότητες και λειτουργίες. Ο ιστότοπος δημιουργήθηκε έτσι ώστε να μπορείτε να διαχειρίζεστε σωστά και εύκολα τις καθημερινές σας εργασίες και δραστηριότητες

Help Section

Welcome to Task Manager! Use the New Task button to quickly create tasks by entering a name, due date, priority, category, status, and optional description. Organize your work efficiently by assigning categories and priorities, and keep track of your progress using task statuses.

Use the Filter button to refine tasks by priority, status, or category, and sort them by due date or priority. The Settings menu allows you to switch between light and dark mode and customize the task border color. Access analytics and more information through the sidebar for better task management and insights.

Καλώς ήρθατε στο Task Manager! Χρησιμοποιήστε το κουμπί Νέα Εργασία για να δημιουργήσετε γρήγορα εργασίες εισάγοντας ένα όνομα, ημερομηνία λήξης, προτεραιότητα, κατηγορία, κατάσταση και προαιρετική περιγραφή. Οργανώστε τη δουλειά σας αποτελεσματικά αναθέτοντας κατηγορίες και προτεραιότητες και παρακολουθήστε την πρόοδο των εργασιών μέσω των καταστάσεων τους.

Χρησιμοποιήστε το κουμπί Φίλτρο για να περιορίσετε τις εργασίες κατά προτεραιότητα, κατάσταση ή κατηγορία και να τις ταξινομήσετε κατά ημερομηνία λήξης ή προτεραιότητα. Το μενού Ρυθμίσεις σας επιτρέπει να εναλλάσσετε ανάμεσα σε φωτεινή και σκοτεινή λειτουργία και να προσαρμόζετε το χρώμα των περιγραμμάτων των εργασιών. Αποκτήστε πρόσβαση σε αναλυτικά στοιχεία και περισσότερες πληροφορίες μέσω της πλαϊνής μπάρας για καλύτερη διαχείριση και εποπτεία των εργασιών.
