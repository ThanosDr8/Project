Task Manager User Manual


1. Overview


The Task Manager is a web application designed to help you create, manage, and organize tasks efficiently. With features like task creation, filtering, sorting, priority management, and analytics, you can keep track of personal or professional work in one place.

---

2. Layout


The interface consists of three main sections:

2.1 Header


The header provides navigation and quick access tools:

Left Section:
Menu Button: Opens the sidebar containing analytics and information/help.

Middle Section:
Search Bar: Search tasks by name.
Filter Button: Opens a modal to filter tasks by priority, status, or category.
View Type Button: Toggle between list and tile views of tasks (icon-based).

Right Section:
Sign In Button: Opens the login modal.
Settings Button: Access settings for themes and interface customization.

---

2.2 Task Section


This is the main workspace for task management:

New Task Button: Opens a modal to create a new task.
Task List: Displays all tasks added by the user. Tasks can be filtered or sorted.

---

2.3 Website Information


Displays a welcoming message and general information about the app.
Contains the app logo and a brief description.

---

2.4 Sidebar


Accessible via the menu button:

Analytics: Redirects to the analytics page to view task statistics.
Information/Help: Redirects to a help page with instructions and FAQs.

---

3. Modals


3.1 New Task Modal


Use this to create new tasks:

1. Task Name: Enter the name of the task.
2. Due Date: Set a due date for completion.
3. Priority: Choose High, Medium, or Low.
4. Category: Assign a category (Work, House Work, School Work, Shopping, Hobbies, Other).
5. Status: Choose Open, In Progress, or Done.
6. Description: Optional detailed description (Max 300 characters).
7. Submit: Adds the task to your task list.

---

3.2 Filter Modal


Use this modal to filter and sort tasks:

1. Priority Filters: High, Medium, Low.
2. Status Filters: Open, In Progress, Done.
3. Category Filters: Work, House Work, School Work, Shopping, Hobbies, Other.
4. Sort By:
Due Date (Ascending/Descending)
Priority (Ascending/Descending)
5. Apply Button: Applies selected filters and sorting to the task list.

---

3.3 Settings Modal


Customize your Task Manager interface:

1. Dark/Light Mode Toggle: Switch between dark and light themes.
2. Border Color Picker: Select a color for task borders.

---

3.4 Sign In / Account Modal


Sign in to personalize your task management:

1. Username: Enter your username (max 12 characters).
2. Password: Enter your password.
3. Sign In Button: Logs you into the system.
4. Cancel Button: Closes the modal without logging in.

---

4. Task List Actions


Once tasks are created:

Tasks appear under the Task List section.
Tasks can be filtered, sorted, and viewed in tile or list mode.
Each task displays its name, due date, priority, category, and status.

---

5. Sidebar Navigation


Click the menu button to open the sidebar.
Close Sidebar: Click the X button or anywhere outside the sidebar.
Options in the sidebar:
Analytics: Visual overview of tasks.
Information/Help: Guides and FAQs.

---

6. Icons & Buttons Reference


Icon/Button | Function
---------------- | -----------------------------
Menu (hamburger) | Opens sidebar
Filter (funnel) | Opens filter modal
View Type (tile) | Switch between list/tile view
Sign In (user) | Opens login modal
Settings (gear) | Opens settings modal
New Task (plus) | Opens new task modal
Close (cross) | Closes modals or sidebar
Apply Filters | Applies filters to task list

---

7. Tips & Best Practices


1. Use categories to organize tasks by type.
2. Set priority and due dates to stay on top of deadlines.
3. Regularly apply filters to focus on important tasks.
4. Use analytics to track progress and task completion trends.
5. Customize colors and theme in settings to match your workflow preference.
